# ElectronAdbClient

## これはなに？

Electronを利用したadb（Android Debug Bridge）をGUI操作するクライアントアプリです。
現時点では

- 接続中のデバイス確認
- apkのインストール
- アプリのアンインストール
- アプリデータのクリア

ができます。

詳細は技術書典4（2018.04.22開催）にて頒布される技術書
『Android開発は人生で大切な全てのことが詰まってる かも知れない本　第2話』
をご参照ください。
